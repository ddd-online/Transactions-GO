# Design Context

## Design Context

### Users
个人桌面记账用户 - 在固定设备上专注记账，需要清晰查看财务数据趋势和图表分析。

### Brand Personality
**沉稳专业型** (Steady Professional) - 类似高端财经杂志（如《经济学人》、《华尔街日报》），数据呈现清晰专业，信息密度与可读性平衡。

### Aesthetic Direction
Editorial / Financial Magazine 风格：
- 强调数据可视化的清晰度和权威感
- 使用醒目的数字和图表作为视觉焦点
- 层次分明的排版，信息模块化呈现
- 双主题支持（亮色/暗色）

### Design Principles
1. **数据优先** - 图表和数据是最重要的，不要让装饰元素干扰数据呈现
2. **清晰的视觉层级** - 使用大小、重量、间距创造信息层次
3. **克制的优雅** - 高端不张扬，专业但有温度
4. **动效服务于认知** - 动画帮助用户理解数据，而非装饰
5. **双向沉浸** - 亮色和暗色主题都需要精心设计，不能只是简单的颜色反转
