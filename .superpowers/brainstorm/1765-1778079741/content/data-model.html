数据库迁移

ALTER TABLE tbl_billadm_transaction_record
ADD COLUMN key_event_date VARCHAR(10) DEFAULT '';

GORM AutoMigrate 会在启动时自动添加此列。

关联逻辑（service 层）

Link（关联）：

  1. 根据 transactionId 查到交易记录
  2. 设置 key_event_date = date，保存
  3. 检查该 date 是否存在 KeyEvent，不存在则创建空记录（仅 date，title/content/color 为空）

Unlink（解除）：

  1. 根据 transactionId 查到交易记录
  2. 设置 key_event_date = ""，保存
  3. 不删除 KeyEvent（即使该事件下已无关联交易）
