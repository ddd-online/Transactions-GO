设计总览：消费记录关联关键事件

数据模型 + API + 前端交互

1. 数据模型

TransactionRecord 新增字段：

字段 类型 说明
key_event_date VARCHAR(10) 关联的关键事件日期，可为空。非空表示已关联至该日。

2. 后端 API

方法 路径 说明
POST /api/v1/tr/link 关联：{ transactionId, date }，若 KeyEvent 不存在则自动创建空记录
POST /api/v1/tr/unlink 解除关联：{ transactionId }
GET /api/v1/tr/linked/:date 查询某日期所有已关联交易记录

3. 前端改动点

文件 改动
TransactionRecordTable.vue 操作列新增"关联"按钮，emit 新事件
TransactionRecordView.vue 处理关联事件，弹出日期选择器，调用 link API
KeyEventView.vue 详情弹窗加 Tabs（详情 / 关联交易），新 Tab 内嵌简化交易表格
keyEventStore.ts 新增 fetchLinkedTransactions action
functions.ts / api/* 新增 link/unlink/fetchLinked API 封装

4. 交互流程

关联流程（交易列表侧）

↓ 点击"关联"

选择日期
